# Requisitos

## Como preencher

Liste aqui os requisitos do seu projeto. Você pode:

1. **Preencher manualmente** — edite este arquivo diretamente
2. **Deixar que o VIBECODE preencha** — durante as conversas, os agentes extrairão os requisitos e os registrarão aqui
3. **Usar o assistente interativo** — execute `hiskra-code init` novamente para ser guiado

> 💡 **Dica:** Se não preencher agora, os agentes preencherão automaticamente
> conforme você for descrevendo o que deseja construir.
