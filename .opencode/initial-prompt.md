# Prompt Inicial

> 🎯 **O que você deseja construir?**
>
> Descreva aqui o seu projeto com o máximo de detalhes possível.
> Quanto mais detalhado, melhor os agentes poderão te ajudar.
>
> **Exemplos:** "Um e-commerce de roupas", "Um sistema de gestão escolar",
> "Uma API para agendamento de consultas", "Um aplicativo de delivery", etc.
>
> Inclua informações como:
> - Público-alvo
> - Problema que resolve
> - Funcionalidades principais
> - Tecnologias desejadas (se houver)
> - Prazo ou urgência

---

*Preencha este arquivo manualmente ou deixe que o VIBECODE o preencha
com base nas suas respostas durante a conversa.*
